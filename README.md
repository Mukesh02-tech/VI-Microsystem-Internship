# VI Microsystem Internship
This repository contains the projects, experiments, and assignments completed during my **Embedded Systems Internship** at **VI Microsystems Pvt. Ltd., Chennai** (June 2025).

The projects were developed using **Raspberry Pi Pico (RP2040)** and the **8051 Microcontroller**, covering fundamental and intermediate embedded systems concepts through hands-on implementation.

## Development Environment

### Raspberry Pi Pico
- Programming Language: MicroPython
- IDE: Thonny IDE

### 8051 Microcontroller
- Programming Language: Embedded C
- IDE: Keil µVision
- Flashing Tool: Flash Magic
- Serial Communication: PuTTY

## Topics Covered

- LED Blinking
- Push Button Interfacing
- Seven Segment Display
- 16x2 LCD Interfacing
- Buzzer Control
- Timer Programming
- Interrupt Programming
- External Interrupts
- ADC Interfacing
- Ultrasonic Sensor
- IR Sensor Interfacing
- DIP Switch Interface
- UART Communication
- Serial Terminal Communication using PuTTY
- GPIO Programming
- Embedded C Programming
- MicroPython Programming
- Hardware Interfacing and Debugging
